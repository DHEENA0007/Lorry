Transport & Fleet Management Application – Complete Draft
1. Project Overview

This Transport Management System is a full-scale logistics and fleet operations platform designed to manage lorries, drivers, bookings, trip execution, and financial tracking. The system consists of:

Super Admin Web Portal (React + Tailwind v4)

User Mobile Application (Flutter)

Driver Mobile Application (Flutter)

Backend API (Node.js)

Database (SQL – PostgreSQL / MySQL recommended)

Google Routes API Integration for route mapping, tracking, and analytics

The application supports real-time lorry tracking, smart booking (similar to Rapido/Ola model), route-based cost estimation, trip budgeting, and post-trip financial reconciliation.
